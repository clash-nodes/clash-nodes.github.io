详细配置步骤图解

第一阶段：路由器基础设置

步骤1：刷入增强型固件
1. 访问路由器管理界面（192.168.1.1）
2. 导航至【高级】-【管理】-【固件升级】
3. 上传下载好的Merlin固件（如R7000380.700.chk）
4. 等待约5分钟完成刷机

专业提示：刷机过程中切勿断电，建议使用有线连接操作

步骤2：开启必要功能
1. 启用JFFS分区（用于存储脚本文件）：
【系统管理】-【系统设置】-【Format JFFS partition at next boot】选"是"
2. 开启SSH远程管理：
【系统管理】-【系统设置】-【Enable SSH】选择"LAN only"

第二阶段：科学上网核心配置

方案A：Shadowsocks透明代理（推荐）
1. 通过SSH连接路由器（用户名为admin，密码为管理密码）
2. 安装Entware环境：
bash opkg update && opkg install shadowsocks-libev 3. 配置ss-redir：
json { "server":"your_server_ip", "server_port":443, "password":"your_password", "method":"aes-256-gcm", "timeout":300, "fast_open":true } 4. 设置iptables规则实现全局代理：
bash iptables -t nat -A PREROUTING -p tcp -j SHADOWSOCKS

方案B：OpenVPN客户端配置
1. 下载服务商提供的.ovpn配置文件
2. 在路由器界面进入【VPN】-【OpenVPN客户端】
3. 导入配置文件并启用"自动重连"选项
4. 设置策略路由（仅特定设备走VPN）：
bash ip route add default via $VPN_GW dev tun0 table 100 ip rule add fwmark 0x1 table 100

第三阶段：智能分流设置

为避免国内网站也走代理导致速度下降，需要配置智能分流：

  1. 安装dnsmasq-full替代原DNS服务：
    bash opkg install dnsmasq-full
  2. 配置国内IP段直连：
    bash wget -O /jffs/configs/china_ip.txt https://raw.githubusercontent.com/17mon/china_ip_list/master/china_ip_list.txt
  3. 设置分流规则：
    bash for ip in $(cat /jffs/configs/china_ip.txt); do iptables -t mangle -A SHADOWSOCKS -d $ip -j RETURN done

高级优化技巧

1. 双WAN负载均衡

通过USB 3.0接口接入4G网卡作为备用线路，当主线路故障时自动切换：
bash iptables -A PREROUTING -t mangle -j WANLOADBALANCE

2. 定时重启策略

每周日凌晨3点自动重启：
bash cru a RouterReboot "0 3 * * 0 /sbin/reboot"

3. 流量监控

安装vnStat实时监控流量使用：
bash opkg install vnstat vnstat -l -i tun0

常见问题解决方案

Q1：连接VPN后网速明显下降
A：尝试以下步骤：
1. 更换加密方式（如将AES-256改为CHACHA20）
2. 测试不同服务器节点
3. 检查是否开启硬件加速（【高级】-【LAN配置】）

Q2：部分设备无法科学上网
A：检查：
1. 设备是否被加入了策略路由例外列表
2. 本地DNS是否被污染（建议改用DoH）

Q3：如何实现Netflix跨区观看
A：需要：
1. 确认VPN支持流媒体解锁
2. 在路由器设置DNS覆写：
bash nvram set custom_dns="8.8.8.8,1.1.1.1" nvram commit
